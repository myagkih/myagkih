# Проект "Калькулятор для математического анализа"

## Инструкция по установке

В командной строке:

- На Linux и Mac:

`git clone https://github.com/myagkih/myagkih.git && cd myagkih && python3 -m venv venv && source venv/bin/activate && pip install -r requirements.txt && python3 main.py`

- На Windows:

`git clone https://github.com/myagkih/myagkih.git && cd myagkih && python -m venv venv && venv/bin/activate && pip install -r requirements.txt && python main.py`

## Возможности проекта

1. Вычисление производных n-ого порядка
2. Вычисление частных производных
3. Вычисление неопределенных интегралов
4. Вычисление определенных интегралов
5. Потроение графика функции
